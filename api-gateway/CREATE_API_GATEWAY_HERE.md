## API Gateway

How to run api gateway

> Make sure that you run **JSON Server** first.

```
cd api-gateway
yarn dev
```

api gateway will run on port 4000

### API Spec

#### 1. Request trip list
User can request the trip list with the limitation of data per page

```
GET /api/v1/trip
```
##### Parameters

| Name        | Type        | Default | Description |
| ----------- | ----------- | ------- | ----------- |
| page        | number      |    1    | page number |
| limit       | number      |    2    | number of display list per page |
| keyword     | string      |         | searching keyword |

##### Example
```
GET /api/v1/trip?page=1&limit=2&keyword=ธรรมชาติ
```
